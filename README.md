# Restaurant Website Layouts – Task 4 Web Application Development

## Overview

This project demonstrates four different web layout approaches for a fictional restaurant website:

* Fixed Layout
* Fluid Layout
* Adaptive Layout
* Responsive Layout

The website includes a header, navigation menu, main content area displaying menu items and specials, and a footer.

## Purpose

The purpose of this project is to compare how different layout techniques behave across desktop, tablet, and mobile devices and to evaluate their effectiveness in creating accessible and user-friendly web designs.


## Layout Implementations

### Fixed Layout

Uses fixed pixel dimensions for all page elements. The layout remains the same regardless of screen size.

### Fluid Layout

Uses percentage-based widths that allow content to scale according to the viewport size.

### Adaptive Layout

Uses predefined layouts that change at specific screen-size breakpoints.

### Responsive Layout

Combines fluid sizing and media queries to create a flexible layout that adapts smoothly to different devices.
